## RabbitHoles AI: Break Free from Linear Chats

[RabbitHoles AI](https://www.rabbitholes.ai/) is an app to have AI conversations on an Infinite canvas. Each node on the canvas is a conversation.

## What It Does

Imagine a digital workspace where:
- Multiple AI chats live on one screen
- Conversations can branch and connect
- You control your research flow

## Key Features

- Switch between AI models instantly
- Add PDFs, videos to your workspace
- Create unlimited project canvases
- Use your own API keys
- One-time purchase, lifetime access

## Why It's Different

Traditional chats limit your thinking. RabbitHoles AI:
- Follows natural thought patterns
- Lets you explore topics deeply
- Prevents context loss
- Works for researchers, writers, students
